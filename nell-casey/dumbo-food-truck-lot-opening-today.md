<p>The <a href="https://web.archive.org/web/20130527055529/http://gothamist.com/2013/04/19/food_truck_lot_coming_to_dumbo.php">new DUMBO food truck lot</a> we told you about last week debuts today. <a href="https://web.archive.org/web/20130527055529/https://www.facebook.com/biandang">Bian Dang</a> owner Thomas Yang emailed this morning to let us know about the soft launch and <a href="https://web.archive.org/web/20130527055529/https://www.facebook.com/domotaco">Domo Taco</a> has already set up shop in the dainty space.</p>

<p>The fusion taco truck will be joined by <a href="https://web.archive.org/web/20130527055529/http://www.hibachiheaven.com/">Hibachi Heaven</a> and <a href="https://web.archive.org/web/20130527055529/https://www.facebook.com/fishingshrimp">Fishing Shrimp</a> for the inaugural encampment. We also have word that <a href="https://web.archive.org/web/20130527055529/https://www.facebook.com/biandang">Bian Dang</a> will be vending on Wednesday; follow <a href="https://web.archive.org/web/20130527055529/https://twitter.com/dumbolot">@dumbolot</a> on Twitter for updates about which trucks will be participating. This is <a href="https://web.archive.org/web/20130527055529/http://gothamist.com/2012/10/24/these_japanese_nacho_tots_are.php">going to be dangerous</a>!</p>
