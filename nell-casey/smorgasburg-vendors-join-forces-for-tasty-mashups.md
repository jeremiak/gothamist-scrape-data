<p>A few weeks ago we were dreaming about these <a href="https://web.archive.org/web/20150509140836/http://gothamist.com/2014/01/06/these_fun_ramen_burger_mashups_at_s.php#photo-1">fun Ramen Burger mashups</a>, hoping they&apos;d become a reality. Now we have official word that not only will we be able to nosh on <a href="https://web.archive.org/web/20150509140836/http://www.goramen.com/2014/01/smorgmash.html">Adobo Shack Ramen Burgers</a> but also that the spirit of collaboration has <a href="https://web.archive.org/web/20150509140836/http://gothamist.com/2013/12/01/photos_inside_the_new_indoor_smorga.php">permeated all of Smorgasburg</a>, resulting in many more mashups for munching.</p>

<p>Starting this weekend, 14 vendors will be teaming up for Smorg Mashups, with plans to add more if this debut is a success. Swing by Saturday and Sunday for...</p>

<ul>
<li><a href="https://web.archive.org/web/20150509140836/http://www.goramen.com/">Ramen Burger</a> buns with <a href="https://web.archive.org/web/20150509140836/http://adoboshack.wordpress.com/">Adobo Shack</a> pork belly</li>
<li><a href="https://web.archive.org/web/20150509140836/http://www.regalvegan.com/site/">Regal Vegan</a> Faux Gras (vegan foie gras) inside a <a href="https://web.archive.org/web/20150509140836/http://tacotruck.marketmymarket.com/">Takumi Taco</a> tempura taco shell</li>
<li><a href="https://web.archive.org/web/20150509140836/http://asiadognyc.com/">Asia Dog</a> kimchi mixed with <a href="https://web.archive.org/web/20150509140836/http://milktrucknyc.com/">Milk Truck</a> mac and cheese</li>
<li><a href="https://web.archive.org/web/20150509140836/https://www.beehiveoven.com/">Beehive Oven</a> biscuit smothered in <a href="https://web.archive.org/web/20150509140836/https://www.facebook.com/sundaygravy">Sunday Gravy</a> tomato sauce/&#x201C;gravy&quot;</li>
<li><a href="https://web.archive.org/web/20150509140836/http://www.saludnyc.com/">Salud</a> spicy hot chocolate given a kick with <a href="https://web.archive.org/web/20150509140836/http://www.brooklynroasting.com/">Brooklyn Roasting Co.</a> coffee</li>
<li><a href="https://web.archive.org/web/20150509140836/http://www.monksmeats.com/">Monk&apos;s Meats</a> bulgogi seitan stuffed in <a href="https://web.archive.org/web/20150509140836/http://www.bitesizekitchen.com/">Bite Size Kitchen</a> bao buns</li>
<li><a href="https://web.archive.org/web/20150509140836/http://palenquehomemadecolombianfood.com/">Palenque</a> arepas slathered with <a href="https://web.archive.org/web/20150509140836/http://www.chedbred.com/">Jack&apos;s Chedbred</a> chili</li>
</ul>

<p>Now we just need a <a href="https://web.archive.org/web/20150509140836/http://gothamist.com/2013/11/26/smorgasburg_1.php">Bruffin</a> stuffed with lobster claws and our lives will be complete.</p>
